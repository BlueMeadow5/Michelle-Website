Headers

To make a header, you use the < h > tag. Alone, the < h > tag will not do anything, so you need to add a number after it. The number determines the size of the header. The smaller the number, the bigger the header. The number can only be from 1 to 6. Also, you need to close the header with a < /h > tag.

An example of a header is: < h4 > This is a header < /h4 >

It will look like this:

This is a header


Paragraph Attributes

A paragraph attribute is an attribute that is used for paragraphs. It cannot have a number in it, but it has to be a ending < /p >.

An example of a paragraph attribute is: < p > This is a paragraph < /p >

This is a paragraph

Lists

HTML supports two main types of lists: ordered (numbered) and unordered (bullet points). Lists are useful for organizing information in a structured way.

Unordered Lists

An unordered list is created using the < ul > tag. Each list item is wrapped in < li > tags.

Example:
< ul >
< li >Item 1< /li >
< li >Item 2< /li >
< /ul >
It looks like this:
  • Item 1
  • Item 2

Ordered Lists

An ordered list uses the < ol > tag instead of < ul >. It automatically numbers the items.

Example:
< ol >
< li >First item< /li >
< li >Second item< /li >
< /ol >
It looks like this:
  1. First item
  2. Second item

Links

Links, created with the < a > tag, allow you to connect different web pages or resources. The href attribute specifies the URL the link points to.

Example: < a href="https://www.example.com">Visit Example.com< /a >

It looks like this: Visit Example.com

When clicked, this link would take you to example.com in your web browser.

Images

Images can be added to your webpage using the < img > tag. It's a self-closing tag, meaning it doesn't need a separate closing tag.

The < img > tag requires two main attributes:
  • src: specifies the path to the image file
  • alt: provides a text description of the image for accessibility

Example: < img src="image.jpg" alt="Description of image" >
